## はじめに

**PeoriaSlider**は、ゴルフコンペの順位をリアルタイムでシミュレーションできるWebアプリです。

### このアプリでできること

- 新ペリア（ダブルペリア）方式でのハンディキャップ自動計算
- 係数や隠しホールを変えたときの順位変動をリアルタイムで確認
- 「もし係数が0.8じゃなくて0.7だったら順位はどうなっていた？」といったシミュレーション
- スマートフォンでの操作に対応

### 対象ユーザー

- コンペ幹事の方：集計作業の効率化、表彰式での演出
- コンペ参加者の方：自分の順位や計算根拠の確認

## 画面の見方

### ヘッダー部分

画面上部には以下の情報がリアルタイムで表示されます：

| 表示項目 | 説明 |
|---------|------|
| **係数** | 現在設定されている計算係数（0.10〜1.00） |
| **隠しH** | 選択中の隠しホール数 |
| **参加者** | スコア入力完了者数 / 登録者数 |

### タブナビゲーション

画面には3つのタブがあります：

1. **設定** - 計算ルールの設定
2. **スコア入力** - プレイヤー登録とスコア入力
3. **順位表** - 計算結果の表示

---

## 基本的な使い方

### ステップ1：プレイヤーを登録する

1. 「スコア入力」タブを開く
2. プレイヤー名を入力して「追加」ボタンを押す
3. 必要な人数分、繰り返す

### ステップ2：スコアを入力する

1. 各プレイヤーの18ホール分のスコアを入力
2. 黄色くハイライトされているホールは「隠しホール」です
3. すべてのホールにスコアを入力すると、自動的に順位が計算されます

### ステップ3：順位を確認する

1. 「順位表」タブを開く
2. 順位、名前、グロス、ハンディキャップ、ネットスコアが表示されます

### ステップ4：シミュレーションを楽しむ

1. 「設定」タブを開く
2. 係数スライダーを動かすと、リアルタイムで順位が変わります
3. 隠しホールを変更しても順位が再計算されます

## 設定タブの使い方

### 計算係数（メイン機能）

**これがこのアプリの核心機能です！**

- スライダーを左右に動かして係数を変更します
- 範囲：0.10 〜 1.00（0.05刻み）
- 標準的な新ペリア方式では **0.80** を使用します
- 係数を変えると、順位表がリアルタイムで更新されます

**使い方の例：**
> 表彰式で「もし係数が0.85だったら...」とスライダーを動かすと、
> 順位がその場で入れ替わる様子を見せることができます。

### 隠しホール選択

- ボタンをタップして隠しホールを選択/解除します
- 黄色＝選択中、白＝未選択
- 「ランダム12H」ボタン：OUT/INから各6ホールをランダムに選択

**新ペリア標準：** 18ホール中12ホールを隠しホールとして使用

### Par設定

「Par設定」をタップすると展開されます。

- 各ホールのPar（3, 4, 5）を設定できます
- 通常は変更不要です（デフォルトで合計Par72）

### 詳細設定

「詳細設定」をタップすると展開されます。

| 設定項目 | 説明 | 標準値 |
|---------|------|-------|
| **隠しホール重み** | 隠しホールスコアに掛ける倍率 | 1.5（新ペリア） |
| **ダブルパーカット** | 各ホールの上限をPar×2にする | ON |
| **HDCP上限** | ハンディキャップの最大値 | 無制限 |
| **端数処理** | 小数点の扱い方 | 四捨五入 |

#### 隠しホール重みについて

| 方式 | 重み | 説明 |
|-----|------|------|
| 新ペリア | 1.5 | 12ホール × 1.5 = 18ホール分として計算 |
| ペリア | 3.0 | 6ホール × 3.0 = 18ホール分として計算 |

#### ダブルパーカットとは

各ホールのスコアに上限を設けるルールです。

例：Par4のホールで10打叩いた場合
- ダブルパーカットON → 8（Par4×2）として計算
- ダブルパーカットOFF → 10のまま計算

大叩きによる極端なハンディキャップを防ぐ効果があります。

#### HDCP上限について

ハンディキャップの最大値を制限する設定です。以下から選択できます：

| 選択肢 | 説明 |
|-------|------|
| **無制限** | 制限なし（デフォルト） |
| **36（男性標準）** | 公式ハンディキャップの男性上限 |
| **40（女性標準）** | 公式ハンディキャップの女性上限 |
| **72（ダブルカット）** | コースPar分まで許容 |

**例：** HDCP上限を36に設定した場合、計算結果が40であっても36として扱われます。

## スコア入力タブの使い方

### プレイヤーの追加

1. 入力欄にプレイヤー名を入力
2. 「追加」ボタンを押す（またはEnterキー）

### プレイヤー名の変更

- プレイヤー名をタップすると編集できます

### プレイヤーの削除

- プレイヤー名の右にある「×」ボタンを押す

### スコア入力

- 各ホールの数字入力欄にスコアを入力
- **黄色いマス**は隠しホールを示しています
- EnterキーまたはTabキーで次のホールに移動できます

### 表示される情報

| 行 | 内容 |
|----|------|
| ホール番号 | 1〜9（OUT）、10〜18（IN） |
| Par | 各ホールのPar値 |
| 打 | 入力するスコア |
| 計 | OUT/INそれぞれの合計 |
| TOTAL | 18ホール合計（グロススコア） |

## 順位表タブの見方

### 表の列

| 列 | 内容 |
|----|------|
| **順位** | 計算された順位 |
| **名前** | プレイヤー名 |
| **Gross** | 18ホール合計スコア（素の打数） |
| **HDCP** | 計算されたハンディキャップ |
| **Net** | Gross − HDCP（この値で順位決定） |

### 順位のハイライト

| 色 | 意味 |
|----|------|
| 黄色（金） | 1位 |
| グレー（銀） | 2位 |
| オレンジ（銅） | 3位 |
| 紫 | ブービー賞（BB） |
| 赤 | 最下位 |

### 順位変動の表示

係数や隠しホールを変更すると：
- **↑** 順位が上がった
- **↓** 順位が下がった

## データの保存・読み込み

### 自動保存

入力したデータは**自動的にブラウザに保存**されます。
- ページを閉じても、次回開いたときにデータが復元されます
- 同じブラウザ・同じデバイスでのみ有効です

### エクスポート（書き出し）

データをファイルとして保存したい場合：

1. 画面下部の「エクスポート」ボタンを押す
2. JSONファイルがダウンロードされます
3. このファイルを保管しておけば、後から読み込めます

**活用例：**
- 過去のコンペデータを保管
- 別のデバイスにデータを移動
- バックアップとして保存

### インポート（読み込み）

保存したファイルを読み込む場合：

1. 画面下部の「インポート」ボタンを押す
2. 保存しておいたJSONファイルを選択
3. データが復元されます

### クリア（データ削除）

すべてのデータを削除する場合：

1. 画面下部の「クリア」ボタンを押す
2. 確認ダイアログで「OK」を押す
3. すべてのプレイヤーと設定が初期化されます

**注意：** クリアしたデータは元に戻せません。必要であれば先にエクスポートしてください。

## 計算方式について

### ハンディキャップの計算式

```
HDCP = (隠しホールスコア合計 × 重み − コースPar) × 係数
```

### 計算例

**条件：**
- 隠しホールのスコア合計：60打
- 隠しホール重み：1.5（新ペリア）
- コースPar：72
- 係数：0.80

**計算：**
```
HDCP = (60 × 1.5 − 72) × 0.80
     = (90 − 72) × 0.80
     = 18 × 0.80
     = 14.4
```

この場合、ハンディキャップは **14.4** となります。

### ネットスコアの計算

```
Net = Gross − HDCP
```

例：グロス92、HDCP14.4の場合
```
Net = 92 − 14.4 = 77.6
```

### 順位決定ルール

同じネットスコアの場合、以下の優先順位で順位を決定します：

1. **ネットスコア**が少ない方が上位
2. 同点の場合：**ハンディキャップ**が少ない方が上位（実力上位）
3. それも同点の場合：**グロススコア**が少ない方が上位

## よくある質問

### Q: スマートフォンで使えますか？

**A:** はい、スマートフォンのブラウザで使用できます。iPhone、Androidどちらにも対応しています。

### Q: インターネット接続は必要ですか？

**A:** 最初にページを開くときは必要ですが、一度読み込めばオフラインでも使用できます。

### Q: データは他の人に見られますか？

**A:** いいえ。データはお使いのブラウザ内にのみ保存されます。サーバーには送信されません。

### Q: ブラウザを変えるとデータはどうなりますか？

**A:** 別のブラウザにはデータは引き継がれません。「エクスポート」機能でファイルを保存し、「インポート」で読み込んでください。

### Q: 係数を変えると何が変わりますか？

**A:** 係数はハンディキャップの大きさに影響します。
- 係数が大きい（1.0に近い）→ ハンディキャップが大きくなる → 初心者有利
- 係数が小さい（0.1に近い）→ ハンディキャップが小さくなる → 上級者有利

### Q: 新ペリアとペリアの違いは？

**A:**

| 方式 | 隠しホール数 | 重み |
|-----|------------|------|
| 新ペリア（ダブルペリア） | 12ホール | 1.5 |
| ペリア | 6ホール | 3.0 |

新ペリアの方が隠しホールが多いため、運の要素が少なく、より実力が反映されやすいと言われています。

### Q: 表彰式でどう使えばいいですか？

**A:** おすすめの演出：

1. まず通常の係数（0.80）で正式な順位を発表
2. 「もし係数が違ったら...」とスライダーを動かす
3. 順位が入れ替わる様子を見せて盛り上げる
4. 「あと少しで順位が変わっていた！」などコメント

## 困ったときは

### データが消えてしまった

- ブラウザのキャッシュをクリアすると、保存データも消えます
- 重要なデータは「エクスポート」で定期的にバックアップしてください

### 順位が計算されない

- すべてのホール（18ホール）にスコアを入力してください
- スコアが0のホールがあると計算されません

### 画面が崩れる

- ブラウザを最新版にアップデートしてください
- ページを再読み込み（リロード）してみてください
